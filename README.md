# VisPrompt Studio

参考画像を選ぶだけで、画像生成AI向けの日本語プロンプトを組み立てられるビジュアルプロンプトビルダーです。

## 機能

- 元資料と個別選定した写真リンクに1対1で対応した59候補から選択
- 全12種類・24カテゴリーから用途に合う参考画像を選択
- カテゴリー別の参考画像を複数選択
- 選択内容からタグの重複を除いてプロンプトを自動生成
- 作りたいもの・テーマの自由入力
- 生成プロンプトの編集とワンタップコピー
- LIVE PROMPTパネルの展開・収納
- スマートフォン／PC対応

## ローカルで確認

ビルドは不要です。任意の静的ファイルサーバーで公開してください。

```bash
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開きます。

## デプロイ

Cloudflare Pagesでリポジトリを接続し、ビルドコマンドを空欄、出力ディレクトリを `/` に設定します。

## 画像について

初期版の参考画像はUnsplash配信画像を使用しています。一般公開・商用化の前に、利用する各画像のライセンスと配信方針を再確認してください。

# VisPrompt Studio

参考画像や選択項目から、画像生成AI向けの日本語プロンプトを組み立てられるビジュアルプロンプトビルダーです。

## 機能

- 要件定義書v2の全13テーマ・110カテゴリー・631アイテムを表示
- 画像は誤対応を避けるため、正確に紐付けできるまで一時的に非表示
- 画像未設定の項目は、画像枠を出さず選択カードとして表示
- テーマ内の全カテゴリーを横断するアイテム検索
- スマホではカテゴリーを横スクロールタブで切り替え
- 前後のカテゴリー移動と現在位置表示
- 制作タイプのアイコンを絵文字ではなく統一トーンのSVGアイコンへ変更
- 意味が伝わりにくいヘッダーのSTEP表示を非表示化
- ファーストビューの見出しサイズを抑え、制作タイプを早く見られるよう調整
- カテゴリー別の項目を複数選択
- 選択中アイテムの一覧表示と選択済みだけ表示
- 選択アイテムだけを一括解除
- 出力スタイルの切り替え（短め／標準／詳しめ）
- 出力形式の切り替え（文章／YAML／YAML + 文章／JSON）
- 初期出力形式はYAML
- YAMLは `prompt` 配下にカテゴリー別アイテムをまとめる簡潔な構造
- `YAML + 文章` では `image_prompt` に画像生成AIへ貼り付けやすい自然文を追加
- コピー操作は1ボタンに統一し、出力形式に応じてコピー対象を自動切り替え
- 避けたい要素を任意入力し、`negative` に反映
- 選択内容からタグの重複を除いてプロンプトを自動生成
- 作りたいもの・テーマの自由入力
- 生成プロンプトの編集とワンタップコピー
- LIVE PROMPTパネルの展開・収納
- スマートフォン／PC対応

## ローカルで確認

ビルドは不要です。任意の静的ファイルサーバーで公開してください。

```bash
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開きます。

## デプロイ

VercelでGitHubリポジトリを接続し、静的サイトとしてデプロイします。

## 画像について

参考画像は、アイテム名との一致を確認できたものから再追加します。一般公開・商用化の前に、利用する各画像のライセンスと配信方針を再確認してください。
